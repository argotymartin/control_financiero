[ ] 14. DOCKERIZAR APP (preparacion migracion)
       - Empaquetar app en contenedor Docker
       - Dockerfile con Python 3.13 + deps sistema (tesseract, dlib, libs)
       - docker-compose.yml para dev local reproducible
       - .dockerignore excluir secrets y archivos pesados
       - Mantener PythonAnywhere funcionando (no romper)
       - Habilita futura migracion a Render/Cloud Run con custom domain
       - Permite dev local identico a prod
       - Tecnologia: Docker + docker-compose
       - Estimado: 1 dia, ~200k tokens
       - Costo: Gratis (Docker CE local)

[ ] 15. MIGRAR HOSTING CON CUSTOM DOMAIN
       - Comprar dominio (~$10 USD/año, ej: .com o .xyz)
       - Migrar de PythonAnywhere a Render/Fly.io/Cloud Run
       - Requiere item 14 (Dockerizar) primero
       - Custom domain gratis en tier free de Render
       - HTTPS automatico
       - URL limpia (sin pythonanywhere.com)
       - Tecnologia: Docker + Render/Fly/Cloud Run + Cloudflare/Namecheap
       - Estimado: 2 dias, ~300k tokens
       - Costo: ~$10-15 USD/año (solo dominio)

[ ] 16. MIGRAR A AWS (alternativa a #15)
       - Opcion A: App Runner (mas simple, managed)
           * Empuja imagen Docker a ECR
           * App Runner corre contenedor con autoscale + HTTPS
           * ~$15-25 USD/mes
       - Opcion B: ECS Fargate (mas control, container serverless)
           * Task definition + service + ALB
           * ~$10-30 USD/mes segun trafico
       - Opcion C: EC2 t2.micro (VM tradicional)
           * Free tier 1 año, despues ~$10 USD/mes
           * Instalar manualmente Flask + Nginx + systemd + certbot
       - Opcion D: Elastic Beanstalk (managed Flask, no docker obligatorio)
           * Deploy con `eb deploy`, EB maneja EC2 + ALB
       - Prerequisitos:
           * #14 Dockerizar (excepto opcion C y D sin docker)
           * Cuenta AWS + IAM user con permisos
           * Route 53 para DNS (o Cloudflare)
           * Certificados ACM para HTTPS custom domain
       - Deps nativas (tesseract, dlib): incluir en Dockerfile via apt-get
       - Supabase sigue externo (sin cambio)
       - Logs: CloudWatch (vs PA console)
       - Secrets: AWS Secrets Manager o Parameter Store
       - Deploy: docker push ECR + App Runner update (~1 comando)
       - Estimado: 1-3 dias primera vez, 600k-1M tokens
       - Costo: $10-30 USD/mes + dominio
       - Trade-off vs #15 Render/Fly: mas complejo, mas caro, mas "profesional"/enterprise
